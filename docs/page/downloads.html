Client Patch 11/21/2001 Akilla
Asheron's Call uses a rotating CRC wheel that calculates checksums for all the packets transferred back and forth. This system has not been cracked yet, and as a result, to use the emulator, we must patch the client.exe back to a copy from May 2001. The IPS Patch for that is here, as well as a simple IPS patcher.

The current patch is for the november client to patch it back to may.
