Welcome to the AC Server Emulator's Website!

The Asheron's Call Server Emulator is a complete alternate server for Asheron's Call. It has very minimal features at the moment, however it is advancing quickly. The team members right now are Akilla and Cibo Matto, the creator of Decal, however I am looking for people to work on packet decoding.

History Note: This emulator is not Sentou's emulator, it is a seperate emulator that I was working on before the original team broke up. Sentou later picked up the code that I was working on and slightly changed it into his current emulator. He is beginning to rewrite it to not use my old source. I neither blame nor fault him for using the source, as far as he knew we had abandoned it. I am now back and continuing development. The original team consisted of KewtSquirrel, Baynah/Narc, and I. I left for 3 months and that's when the team diverged. We do not plan on merging the two emulators, please do not ask.

XP System + Some Bugfixes 1/11/2001 Akilla
Okay, I added a full XP system in. i.e. you can give yourself XP and then raise skills and stats with it. I also added a !setstatinc command so you can give yourself 100000 run and jump and go flying everywhere. Buncha other neat stuff added now.

The known bugs are as follows. 1. There's still some kind of buffer bug in there. I thought I got rid of it, and in any event it's a lot better than earlier betas, but it's still there. 2. There may be an obscure crash bug on disconnection. If you can reproduce it, please tell me via email or the boards. Thanks!

Any other minor or major bugs please post on the boards or email me about. Make sure you delete your old .dat files before using the newest betas. Enjoy!

Merry XMas 12/25/2001 Akilla
Okay, as a nice xmas gift, I'll post a link to the latest beta and some other stuff going on behind the scenes here. First off, you can grab the latest betas here: http://home.earthlink.net/~killaw/ACServer.

This works with the DECEMBER CLIENT. However, I would make a COPY of your current asheron's call directory, and then use the logon client with that.

Latest betas work with the directory server hosted kindly by Jubilee at http://www.forotech.com/scripts/dirserv.exe. To have the connect link on the directory server page work, first run the login client once then exit, so that it can set up the integration in your registry. Then close IE and reopen IE, and the directory server links should work properly.

Enjoy! There are numerous bugs with this version. Most notable is there is a buffer problem, where if the server is sending too much data to the client, the client messes up and ignores all future packets from the server. This is the next major problem we're addressing.

Other than that, it is mostly fully functional. !givexp is working. I believe PvP combat is buggy, however. The worldstore is fixed, and character creation should be working as normal. Bug reports will only be accepted for the december server, not for the old release. I'd like to pretend that one doesn't exist. :-)
